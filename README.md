# RHA Overhead & Hiring Planner (V2)

## Run locally

```bash
npm install
npm run dev
```

Then open the URL printed in your terminal (usually http://localhost:5173).

If you hit errors on Windows, make sure Node.js 18+ is installed and try:
```bash
rd /s /q node_modules
npm cache clean --force
npm install
```

## Build
```bash
npm run build
npm run preview
```
